# じゃんけんサバイバー 🪨✌️✋

> じゃんけんで敵を倒して生き残れ！スコアを守りながらウェーブをクリアするブラウザゲームです。

---

## 🎮 遊び方（基本）

画面に出てくる敵は **グー・チョキ・パー** のどれかです。  
画面下のボタンをタップして「勝てる属性」の弾を撃ち、敵を倒してください。

| あなたの攻撃 | 敵 | 結果 |
|:-----------:|:--:|:----:|
| **勝ち** | 相手に勝つ手 | 2倍ダメージ ✅ |
| **あいこ** | 同じ手 | 1倍ダメージ ⚠️ |
| **負け** | 相手に負ける手 | ダメージなし ❌ |

### じゃんけんの相性

```
グー ✊ → チョキ ✌️ に勝つ
チョキ ✌️ → パー ✋ に勝つ
パー ✋ → グー ✊ に勝つ
```

---

## 📱 操作方法

| 操作 | 内容 |
|------|------|
| 画面下の **✊ グー** ボタン | グーの弾を発射 |
| 画面下の **✌️ チョキ** ボタン | チョキの弾を発射 |
| 画面下の **✋ パー** ボタン | パーの弾を発射 |
| 画面中央上の **💣 BOMB** ボタン | ボムを使用（画面上の雑魚を一掃） |

弾は自動的に近くの敵を追いかけます。

---

## 💯 スコアとライフについて

このゲームでは **スコア＝ライフ** です。

- **スコアが0になるとゲームオーバー**
- 敵を倒すと **スコアが増える**
- 敵が画面下を通り抜けると **スコアが減る**
- ボムを使うと **現在スコアの半分を消費** する

> 💡 スコアが赤く表示されているときは危険なサインです！

## 👾 敵の種類

| 種類 | 見た目 | HP | 通過ペナルティ | 特徴 |
|:----:|:------:|:--:|:------------:|------|
| **通常雑魚** | 小さい円 | ×1 | ×1 | 普通に下へ動く |
| **中型雑魚** | 白枠付き円 | ×2 | ×3 | 少し固く、通過時のダメージが大きい |
| **大型雑魚** | 外リング付き円 | ×5 | ×5 | かなり固く、通過は即死級 |

| **大ボス** | 👑👑 二重王冠・紫オーラ | 超高 | ×3 | 大量召喚スキルに加え、2種のバリアを交互に使う強敵 |

### ボス・大ボスのバリア

| 表示 | 見た目 | 効果 |
|:----:|:------:|------|
| 🛡 GUARD | 青い光輪 | 完全ダメージ無効 |
| ⊘ BARRIER | 白い点線輪（大ボスのみ） | あいこ攻撃が無効化される |

> 💡 バリア中は攻撃が弾かれます。無防備な瞬間を狙って集中攻撃しましょう！

### 大ボスのスキル（約3.5秒ごとに順番に使用）

| スキル | 内容 |
|:------:|------|
| **A** | 1属性の通常雑魚を大量（15〜20体）召喚 |
| **B** | 混合属性の中型雑魚を7〜8体召喚 |
| **C** | あいこ無効の大型雑魚を2〜3体召喚 |

> 💡 大ボスは「無防備」「🛡 GUARD」「⊘ BARRIER」が約1/3ずつ切り替わります。バリアは攻撃しても無意味なので、召喚された雑魚の処理を優先しましょう。

## 💣 ボムについて

- 1ステージに **3回** 使用可能（スキルで増やせる）
- 使用コスト：**現在スコアの半分**
- 効果：画面上の **雑魚をすべて爆破**
- ボス・大ボスには効かない

---

## ✨ スキルショップ

ウェーブをクリアするたびにスキルを **1つだけ** 選べます。  
スキルは **グー・チョキ・パー・汎用** の4列に1枚ずつ表示されます。

### コストについて

同じ列で選ぶたびにコストが **2倍** になります。

```
例：グー列
1回目 → 300pt
2回目（どのスキルでも）→ 600pt
3回目 → 1200pt
```

## 🔥 上手くなるコツ

1. **じゃんけんの相性を優先** — 負け属性を撃つのは無駄。相性を確認して撃とう、特に大ボスが召喚するあいこ無効雑魚に注意。
2. **スコアが高いときはボムを温存** — スコアが半分になるので、スコアが高いときに使うとジリ貧に。強化で「無料ボム」を確保しよう。
3. **スキルは1つの列に集中投資** — コストは列ごとに増えるので、まんべんなく強化がオススメ。
4. **大ボス戦はバリアを見極めよ** — 🛡 GUARD中は攻撃が全て無効、⊘ BARRIER中は勝ち属性のみ有効。無防備の瞬間に一気に削ろう。大型雑魚の通過はスコアが5倍減るので最優先で処理。
5. **守護盾はライフが少ないとき最強** — 被弾直後に無敵を発動（CT5秒）。Lvを上げるほど無敵時間が延びる。画面が青くなったら発動中のサイン。
